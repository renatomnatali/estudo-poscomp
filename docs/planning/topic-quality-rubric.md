# Rubrica de Qualidade dos Tópicos

## Critérios de aprovação
1. Precisão técnica: sem erro conceitual.
2. Cobertura didática: essencial + avançado + exemplos + aplicação.
3. Utilidade para prova: ligação clara com formato POSCOMP.
4. Clareza textual: sem ambiguidade e sem redundância.
5. Verificabilidade: quick-check com explicação correta.

## Escala
1. `critical`: erro técnico ou ausência de seção obrigatória.
2. `major`: explicação confusa, exemplo incompleto, aplicação fraca.
3. `minor`: ajuste de redação, ordem ou detalhe visual.
4. `nit`: microajuste sem impacto de compreensão.

## Gate mínimo para publicação
1. Pelo menos 1 seção `essential`.
2. Pelo menos 1 seção `advanced`.
3. Pelo menos 1 exemplo resolvido.
4. Pelo menos 1 aplicação real.
5. Pelo menos 1 quick-check válido.
